Default Keybindings

Global Menus

File

  • New File
    • meta N
  • Save current buffer
    • meta S
  • Save current buffer as…
    • meta shift S

Edit

  • Undo last recordable action in the current buffer
    • meta Z
  • Redo last recordable action in the current buffer
    • meta shift Z
  • Copy selected text to clipboard from current buffer text area
    • meta C
  • Paste current clipboard contents to the current buffer’s caret position
    • meta V
  • Cut the selected text in the current buffer
    • meta X

View

  • Go to project file tree
    • meta shift 1
  • Go to current buffer
    • meta shift 2
  • Go to current REPL
    • meta shift 3
  • Show/hide file tree
    • meta 1
  • Show/hide REPL
    • meta 2
  • Next buffer tab
    • meta alt RIGHT
  • Previous buffer tab
    • meta alt LEFT
  • Next REPL tab
    • meta alt UP
  • Previous REPL tab
    • meta alt DOWN
  • Close current buffer tab
    • meta W

Source

  • Search for a word in the current buffer via the application REPL search function
    • meta F
  • Search and replace for a word in the current buffer via the application REPL search function
    • meta shift F
  • Search and replace all occurrences of a word in the current buffer via the application REPL search function
    • meta control F
  • Toggle comment for current line of active buffer
    • meta /

REPL

  • Previous REPL entry
    • control UP
  • Next REPL entry
    • control DOWN
  • Next REPL tab
    • meta alt UP
  • Previous REPL tab
    • meta alt DOWN
