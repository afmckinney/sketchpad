Config

Overview

SketchPad supports a number of configuration options from buffer key bindings to the look and feel of most components. All of these preferences are managed by the default.clj file which is created in ~/.sketchpad/default.clj. If the ~/.sketchpad directory does not exist then it will be created along with the default.clj file and the default theme file. The default config file provides access to configuring:

REPL buffer prefs

  • line wrapping
  • buffer styles
    • visible whitespace
    • selection border and fill styles
    • eol markers
    • mark occurrences rendering
    • bracket matching color
    • bracket matching border color
  • auto indent
  • fonts
  • code folding
  • tab size
  • mark occurrences
  • show/hide scroll bars
  • enable hyperlinks
  • tab lines
  • enable code templates
  • anti aliasing
  • animate bracket matching
  • close curly braces
