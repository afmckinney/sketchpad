REPL

SketchPad Application REPL

A major goal for SketchPad is for the application REPL to be a powerful tool for everything from scripting edits, to managing files, to any other IDE related task. The current features include

User namespace sketchpad.user

Utility functions

  • pretty print
    • (pp)
  • print stack trace
    • (stack-trace)
    • (st)
  • pretty print + reflection
    • (preflect java-object)
  • seesaw shortcuts
    • show a seesaw widget’s options
      • (show-opts (current-text-area))
    • show a widgets event types
      • (show-evs widget)

Projects

  • get the current project
    • (current-project)
    • For project map structure checkout the Project description page.
  • create a new Leiningen project
    • (create-project)

Buffers

  • get the current RSyntaxTextArea buffer
    • (current-text-area)
  • get the current form for the active buffers caret position
    • (current-form)
  • current group form based on \newline delimiters
    • (current-group-form)

Search

  • Function call from REPL
    • Search (search "word")
    • Search-replace (search "word" "replace-word")
    • Search-replace-all (search "word" "replace-word")
  • Menu hotkey to add function call to REPL text area and focus REPL
    • Search meta F
    • Search-replace meta shift F
    • Search-replace-all meta control F

Mark occurrences

  • Mark a word -(mark "word-to-mark")
  • Clear all marked occurrences
    • (clear-marks)
    • (mark)
