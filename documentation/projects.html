Projects

The SketchPad project system is base on Leiningen and Clojars. Leiningen is integrated into the application as a core dependency and allows SketchPad to create, build, and run Clojure projects without the need for any other libraries or system dependencies. To begin working on a project you first need to open an existing project or create a new one.

Creating a project

New projects can be created in a few ways:

  • Through the file menu, File -> New Project...
  • or the file tree popup menu New Project...
  • or from the SketchPad REPL with sketchpad.user => (create-project)

This will open the project setup form. The project creation form creates the new project’s project.clj file based on the project param fields. One handy feature incorporated into the creation form is that you can use shorthand names for dependencies available on Clojars.org. The auto completion will query Clojars.org for matching dependencies and display all available versions etc.
